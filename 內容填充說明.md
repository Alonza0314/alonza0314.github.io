## 樣式調整

如果您想調整某些元素的樣式（顏色、大小、間距等），可以修改對應的 CSS 文件：

- 全局顏色和樣式：`src/styles/index.css`
- 主頁樣式：`src/pages/Home.css`
- Header 樣式：`src/components/Header.css`
- Footer 樣式：`src/components/Footer.css`

所有主要顏色都使用 CSS 變量定義，您可以在 `src/styles/index.css` 的 `:root` 選擇器中修改：

```css
:root {
  --primary-color: #6366f1;    /* 主色 - 淡紫色 */
  --secondary-color: #3b82f6;  /* 次色 - 淡藍色 */
  --accent-color: #8b5cf6;     /* 強調色 - 紫色 */
  /* ... 更多顏色 */
}
```

## 注意事項

1. **文字長度**: 請注意文字長度，過長的文字可能會影響頁面佈局
2. **圖片大小**: 大型圖片會影響頁面載入速度，建議壓縮後再使用
3. **連結測試**: 請確保所有外部連結都能正常訪問
4. **文件路徑**: 文件路徑區分大小寫，請確保路徑正確
5. **字符編碼**: 所有文件都使用 UTF-8 編碼，確保中文顯示正常
