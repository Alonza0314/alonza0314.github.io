# 01. React install

- [Node.js](#nodejs)
- [Yarn](#yarn)
- [React](#react)

## Node.js

- Install

    ```bash
    brew install node
    ```

- Check

    ```bash
    node -v
    npm -v
    ```

## Yarn

- Install

    ```bash
    brew install yarn
    ```

- Check

    ```bash
    yarn -v
    ```

## React

- Create Project

    ```bash
    yarn create react-app react --template typescript
    ```

- Dev

    ```bash
    yarn start
    yarn build
    yarn test
    ```
